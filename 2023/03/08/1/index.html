HTML-Day5

文本框和单选框

属性 说明
type 指定元素类型。text，password，radio，checkbox，file，submit，reset，button，image，hidden，默认为text
name 文本框的名称
value 文本框的默认值。type为radio和checkbox时，value为选中时的值
size 文本框的宽度，当type为password或text，size指定的是字符的个数。
maxlength 文本框的最大长度
checked 单选框默认选中

e.g.:

用户注册



男 女 保密

按钮和多选框

e.g.:

爱好 篮球 足球 乒乓球 羽毛球

按钮也是一种特殊的输入框，它的type属性值为button，submit，reset，image。当type为button时，按钮的功能是执行一个JavaScript函数，当type为submit时，按钮的功能是提交表单，当type为reset时，按钮的功能是重置表单，当type为image时，按钮的功能是提交表单，但是按钮的样式是一个图片。比如：

下拉列表框

下拉列表框是一种特殊的输入框，它的type属性值为select，它的功能是从一组选项中选择一个选项。

e.g.:

请选择你喜欢的水果

文本域和文件域

文本域是一种特殊的输入框，它的type属性值为textarea，它的功能是输入多行文本。
