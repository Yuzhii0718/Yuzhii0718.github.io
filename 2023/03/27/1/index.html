本文介绍了 JavaScript ，以及快速入门它的的基本语法和数据类型。

概述

JavaScript 是一种轻量级的编程语言。JavaScript 是可插入 HTML 页面的编程代码。JavaScript 插入 HTML 页面后，可由所有的现代浏览器执行。
JavaScript 是世界上最流行的脚本语言之一。
JavaScript 是一种基于原型、函数先行的语言，是一门多范式的语言，它支持面向对象编程，命令式编程，以及函数式编程。它提供语法来操控文本、数组、日期以及正则表达式等，不支持 I/O，比如网络、存储和图形等，但这些都可以由它的宿主环境提供支持。

JavaScript 与 ECMAScript

JavaScript 是 ECMAScript 的实现。ECMAScript 是由 ECMA（欧洲计算机制造商协会）通过 ECMA-262 标准来定义的一种脚本语言的规范。ECMAScript 的目的是为了使得 JavaScript 在不同的宿主环境中，表现出一致的行为。

数据类型

数值、字符串、图形、音频、视频等都可以被存储在变量中。
js 不区分整数和浮点数，所有数字都是浮点数。

注意：== 和 === 的区别是，== 会进行类型转换，而 === 不会进行类型转换。
NaN 不等于任何值，包括 NaN 本身。只能用 isNaN() 来判断一个值是否为 NaN。尽量避免使用浮点数进行计算，因为浮点数计算会产生误差。

  • 数组

注意：数组的索引是从 0 开始的。
javascript 数组可以存储不同类型的数据。

  • 对象

注意：对象的属性名是大小写敏感的。对象使用花括号 {} 来定义，属性名和属性值用冒号 : 分隔，多个属性之间用逗号 , 分隔。字典是一种特殊的对象，字典的属性名是字符串类型。数组使用方括号 [] 来定义，数组的元素之间用逗号 , 分隔。

注意

  • 严格区分大小写

  • 语句以分号结尾

  • 语句块以花括号 {} 包裹

  • 语句块中的第一条语句不要以分号开头

  • 语句块中的最后一条语句不要以分号结尾

  • 定义变量名称时，使用有意义的名称，不要使用 a、b、c 等无意义的名称，不能使用关键字和保留字作为变量名称，不能以数字开头，不能包含空格和特殊字符（如：!、@、#、$、%、^、&、*、(、)、+、=、{、}、[、]、|、\、:、;、"、'、<、>、?、/、`、~ 等）。
