初试HTML5

H5的基本原理

H5 App 前端开发，是基于 HTML5、CSS3、JavaScript、jQuery、Ajax、JSON、XML 等技术，主要由3部分组成：结构（Structure）、表现（Presentation）、行为（Behavior）。大多数H5应用与CSS相集成，Javascript负责；浏览器的交互行为。HTML5+CSS3+JavaScript相辅相成，紧密合作。

  • HTML 就像是骨架
  • CSS 就像是血肉
  • JavaScript 就像是灵魂

H5的优势

H5的优势主要体现在以下几个方面：

  • H5的跨平台性

  • H5的开发效率

  • H5的开发成本

  • H5的开发周期

  • H5的开发技术

H5的结构元素

  • header 定义文档或者节的页眉

  • section 定义文档中的主体部分

  • nav 定义导航链接的部分

  • aside 定义页面的侧边栏

  • article 定义页面的主要内容

  • footer 定义文档或者节的页脚
