A brief introduction to Linux user and group manage.

Whatâ€™s user and group

  • user: a person who use the computer
  • group: a set of users
  • user and group are used to control the access to the system

root is the super user, it can do anything.

User and group file

  • /etc/passwd: user info

  • /etc/shadow: user password

  • /etc/group: group info

  • /etc/gshadow: group password

  • /etc/login.defs: user default setting

  • /etc/default/useradd: useradd default setting

  • /etc/default/groupadd: groupadd default setting

  • /etc/skel: user default home dir

  • /etc/profile: user default env

  • /etc/bashrc: user default bash env

  • /etc/security/limits.conf: user default resource limit

  • /etc/pam.d: user login pam config

  • /etc/nsswitch.conf: user login nss config

  • /etc/sudoers: user sudo config

  • /etc/sudoers.d: user sudo config

  • /etc/ssh/sshd_config: user ssh config

  • /etc/ssh/ssh_config: user ssh config
