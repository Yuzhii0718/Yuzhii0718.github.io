在导入from PyQt6.QtCharts import QLineSeries, QChart, QChartView时，会报错：

一开始，我认为是 PyQt6-Charts 没有安装成功，但是在 PyCharm 中，我可以看到 PyQt6-Charts 已经安装成功了。随后，我在网上搜索了一下，在简中互联网上，没有找到解决方案，但是在 StackOverflow 上，找到了解决方案。

参考：PyQt6-Charts

参考这篇文章，我先找到了几个疑点：

  1. PyQt6-Charts 是基于 PyQt6 的，而不是 PyQt5。

  2. PyQt6-Charts 可能只创建了文件夹，而没有创建文件。

  3. PyQt6-Charts 对应的是 PyQt6 的哪个版本。

  4. PyQt6-Charts 对应的是 Python 的哪个版本。

对于第一个疑点，我在安装 PyQt6-Charts 时，使用的是 pip install PyQt6-Charts，而不是 pip install PyQt5-Charts，所以，这个疑点是排除的。第二，我在安装 PyQt6-Charts 时，确实只创建了文件夹，而没有创建文件，所以，这个疑点是成立的，所以我从 PyPI 上下载了 PyQt6-Charts 的 whl 安装文件，然后使用 pip install PyQt6_Charts.whl 安装了 PyQt6-Charts。第三，我在安装 PyQt6-Charts 时，没有指定 PyQt6 的版本，所以，这个疑点是成立的，所以我反复尝试了几个 PyQt6 的版本。第四，我尝试创建了多个虚拟环境，使用不同的 Python 版本，包括 conda 环境，但是都没有解决问题。

最后，我发觉问题出自与软件包的兼容性，我使用的是 Python 3.11

requirement.txt

问题可能出在 pyqt6-plugins 上，遂将其卸载，问题解决。
