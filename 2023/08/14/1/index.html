QtQuick & QML

QtQuick 是 Qt 的一个模块，它提供了一套用于创建动态用户界面的 QML 语言，以及一个用于在 C++ 中使用 QML 的 API。

QML(Quantum Markup Language)是一种声明性的语言，它允许用户通过简单的 JavaScript 语法来创建动态的用户界面。QML 语言的语法和 JavaScript 语言的语法非常相似，但是 QML 语言的语义和 JavaScript 语言的语义是不同的。
