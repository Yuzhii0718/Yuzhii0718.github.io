JDBC

JDBC 是 Java Database Connectivity 的缩写，它是 Java 语言操作数据库的统一接口，它定义了一套操作所有关系型数据库的规则，程序员使用 JDBC 接口，可以连接不同的数据库，执行 SQL 语句，获取执行结果。

数据库驱动

JDBC 驱动是一个 Java 类，它实现了 JDBC 接口，不同的数据库需要不同的驱动。例如，MySQL 驱动就是一个 com.mysql.jdbc.Driver 类，它实现了 JDBC 接口，程序通过 JDBC 接口操作 MySQL 数据库时，实际上是调用 MySQL 驱动的代码。

驱动 Driver，驱动程序，通常意义上的驱动程序是指硬件驱动程序，它是一段程序代码，用来控制硬件设备的工作。在 Java 中，驱动程序是指实现了 java.sql.Driver 接口的类，它可以连接到某个具体的数据库。

JDBC 对象解释

每个对象使用的方法都有很多，这里只是列出了一些常用的方法，更多的方法可以查看官方文档。

DriverManager

DriverManager 是驱动管理类，它负责管理 JDBC 驱动，开发者不需要直接和 JDBC 驱动打交道，而是通过 DriverManager 间接和 JDBC 驱动打交道。

Connection

Connection 表示一个数据库连接，它是一个接口，定义了所有数据库连接的公共方法。程序通过 Connection 接口访问数据库。

Statement

Statement 表示一个 SQL 语句的对象，它用于执行一个 SQL 语句。

PreparedStatement

PreparedStatement 表示一个预编译的 SQL 语句的对象，它可以执行动态 SQL 语句。

CallableStatement

CallableStatement 表示一个可以调用数据库存储过程的对象。

ResultSet

ResultSet 表示一个结果集，它封装了从数据库查询到的数据。

SQLException

SQLException 是一个异常类，它是在访问数据库时发生错误时抛出的。
