链表

什么是链表？

链表是一种常见的数据结构，它是由一组节点组成的，每个节点包含两部分：数据域和指针域。

链表于算法中的作用非常大，它可以用来实现栈、队列、哈希表等数据结构。

算法

  • 通俗定义：算法是解决问题的一系列步骤。

  • 狭义定义：算法是对存储数据的操作。

  • 广义定义：也叫泛型算法，是对存储数据的操作，不考虑数据的类型。

我们至少可以通过两种结构来存储数据：数组和链表。

数组的连续性使得它的存取速度非常快，但是数组的大小是固定的，而且数组的插入和删除操作非常耗时。导致了三个问题：

  • 无法存储大量数据

  • 需要一块连续的大量的内存空间

  • 插入和删除操作耗时、效率低

链表的存储方式是非连续的，它的每个节点都包含了数据和指针，指针指向下一个节点，这样就形成了一个链表。

链表的插入和删除操作非常快，但是存取速度非常慢，因为链表的存储方式是非连续的，所以我们无法通过下标来访问链表中的元素。

链表的基本操作

链表的分类

  • 单向链表

  • 双向链表

  • 循环链表

相关概念

  • 头节点：链表中第一个节点

  • 尾节点：链表中最后一个节点

  • 头指针：指向头节点的指针

  • 尾指针：指向尾节点的指针

单向链表是最简单的链表，它的每个节点都包含两个部分：数据域和指针域。

数据域用来存储数据，指针域用来指向下一个节点。

位运算符

位运算符是对二进制位进行操作的运算符，是把数字看作二进制来进行计算。

运算符描述
&按位与运算符，参与运算的两个值，如果两个相应位都为1，则该位的结果为1，否则为0
|按位或运算符，只要对应的二个二进制位有一个为1时，结果位就为1
^按位异或运算符，当两对应的二进制位相异时，结果为1
~按位取反运算符，对数据的每个二进制位取反，即把1变为0，把0变为1
<<左移运算符，运算数的各二进制位全部左移若干位，由"<<"右边的数指定了移动的位数，高位丢弃，低位补0
>>右移运算符，把">>“左边的运算数的各二进制位全部右移若干位，”>>"右边的数指定了移动的位数

NULL

NULL是C语言中的一个宏定义，它的值是0，表示空指针。

二进制中，0表示假，1表示真，而如果二进制全部为0，那么有下面几种情况：

  • 数值 0

  • 字符结束符 ‘\0’

  • 空指针 NULL

NULL 本质也是一个 0，但是它的含义是空指针，而不是数值 0。表示的是内存单元的地址为 0，而不是内存单元中存储的值为 0。

在学科内规定了，以零为编号的内存单元不存放有效数据，所以我们可以用 0 来表示空指针。

杂记

算法的狭义定义

对于狭义的算法，我们可以通过下面的方式来定义：

  • 对存储结构的操作

  • 对不同存储结构，要求的操作是不同的

这说明了算法的两个特点：

  • 算法是依附于存储结构的，不同的存储结构，要求的操作是不同的

  • 算法的效率是依赖于存储结构的

算法的特点

  • 输入：算法具有0个或多个输入

  • 输出：算法至少有1个或多个输出

  • 有穷性：算法在有限的步骤之后会自动结束而不会无限循环，并且每一个步骤可以在可接受的时间内完成

  • 确定性：算法中的每一步都有确定的含义，不会出现二义性

  • 可行性：算法的每一步都是可行的，也就是说每一步都能够执行有限的次数完成

广义算法

广义算法也叫泛型算法，它是对存储结构的操作，不考虑存储结构的类型。

文件

文件是存储在辅助存储器上的数据序列，它是数据的集合。

宏定义

宏定义是一种预处理指令，它是用来定义标识符的，它的作用是在程序中以标识符代替一个字符串。

typedef

typedef是一种类型定义符，它的作用是为一种数据类型定义一个新的名字。
