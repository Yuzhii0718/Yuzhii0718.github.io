接下来的内容是基于 C 语言的数据结构和算法。

什么是数据结构？

我们如何把现实生活中大量而复杂的问题以特定的数据类型和特定的存储结构保存到计算机中，以及在此基础上为实现某个功能而执行的相应操作(这个相应的操作也叫算法)，这就是数据结构要研究的内容。

数据结构 = 数据类型 + 存储结构 + 相关算法

  • 《数据结构 C语言版 (严蔚敏)》

衡量算法的标准

  • 时间复杂度

  • 空间复杂度

  • 难易程度

  • 健壮性

时间复杂度

程序的执行时间 T(n) 与每行代码的执行次数 n 成正比，即：T(n)=O(f(n))，其中 f(n) 表示每行代码执行的次数总和，而 O 表示正比例关系，这个公式的关键是 f(n)。

时间复杂度并不是真正的时间，而是执行次数的计算，因为不同的机器执行相同的代码所需的时间不同，因此用执行次数来比较算法的时间复杂度。

空间复杂度

算法的空间复杂度是指算法需要消耗的空间资源，一般来说，计算机算法是问题规模 n 的函数 f(n)，算法的空间复杂度也因此记做：S(n)=O(f(n))。

链表的定义

链表是一种物理存储单元上非连续、非顺序的存储结构，数据元素的逻辑顺序是通过链表中的指针链接次序实现的。

离散存储结构

离散存储结构是指数据元素的存储地址不连续的存储结构。

  • 定义

    1. n个结点离散分配
    2. 彼此通过指针相连
    3. 每个结点只有一个前驱结点，每个结点只有一个后继结点，首结点没有前驱结点，尾结点没有后继结点。

  • 相关专业术语

    • 首结点：第一个有效结点，它是头指针指向的结点。

    • 尾结点：最后一个有效结点，它的指针域为空。

    • 头结点：为了操作的统一和方便而设立的，放在第一个元素的结点之前，其数据域一般无意义(也可存放链表的长度等信息)，指针域存放第一个元素结点的地址。

    • 头指针：指向头结点的指针变量。

    • 尾指针：指向尾结点的指针变量。
