什么是 Java

Java 是一种跨平台的编程语言，它可以在不同的操作系统上运行，只需要安装对应的 Java 虚拟机（JVM）即可。

特性

  • 简单易学

  • 面向对象

  • 平台无关性

  • 可靠性

  • 安全性

  • 支持多线程

  • 支持网络编程

  • 编译与解释并存

  • 动态扩展

  • 健壮性

  • 开源

  • etc.

应用

  • 桌面应用程序

  • Web 应用程序

  • 企业应用程序

  • 移动应用程序

  • etc.

关于 JDK

JDK 与 JRE 的区别

JDK 是 Java Development Kit 的缩写，它是提供给 Java 开发人员使用的，其中包含了 Java 的开发工具，也包括了 JRE。
JRE 是 Java Runtime Environment 的缩写，它是运行 Java 程序所必须的环境的集合，包含 JVM 标准实现及 Java 核心类库。

JDK 的版本

建议选择LTS版本，LTS是Long Term Support的缩写，即长期支持版本，这种版本的JDK会提供长达几年的支持，而非LTS版本的JDK只会提供半年的支持。
JDK也有不同的发行版本，Oracle JDK是Oracle公司发布的JDK，OpenJDK是开源社区发布的JDK，两者几乎没有区别，Oracle JDK包含了一些Oracle公司的授权代码，OpenJDK完全开源。

我使用了Adoptium Eclipse Temurin 的 JDK，它是OpenJDK的一个发行版本，它的特点是免费、开源、高性能、多平台，它的开发者包括了IBM、Microsoft、Red Hat、SAP等公司，它的前身是AdoptOpenJDK。下载 Adoptium Eclipse Temurin

Java 运行机制

  • 编译型语言：先将源代码编译成机器码，再运行机器码。

  • 解释型语言：运行时才翻译成机器码，每次运行都要翻译一次。

Java 是一种混合型语言，它的源代码是编译成字节码，然后由 JVM 解释执行。

集成开发环境 (IDE)

IDE 是 Integrated Development Environment 的缩写，它是一种软件，可以帮助开发人员提高开发效率，常见的 IDE 有 Eclipse、IntelliJ IDEA、NetBeans、Visual Studio 等。主流的 IDE 都支持 Java 开发，我使用的是 IntelliJ IDEA。因为普遍认为 IntelliJ IDEA 是最好的 Java IDE，而且它的社区版是免费的。恰好，我有 JetBrains 的许可证，可以使用 IntelliJ IDEA 的专业版。

标识符

标识符用来给类、变量、方法等命名，标识符由字母、数字、下划线、美元符号组成，不能以数字开头，不能是 Java 的关键字。

数据类型

Java 是一种强类型语言，每个变量都必须声明其数据类型，Java 的数据类型分为两大类：基本数据类型和引用数据类型。

Java 的基本数据类型都有对应的包装类，包装类是引用数据类型，它们的值可以为 null，而基本数据类型的值不能为 null。最好避免使用浮点数进行比较，因为浮点数的精度有限，可能会出现精度丢失的情况。

类型转换

  • 自动类型转换是指从小范围到大范围的转换，例如：byte 转 short、short 转 int、int 转 long、long 转 float、float 转 double。

  • 强制类型转换是指从大范围到小范围的转换，例如：double 转 float、float 转 long、long 转 int、int 转 short、short 转 byte。

  • 强制类型转换可能会导致精度丢失，例如：double 转 float、float 转 long。

  • 强制类型转换可能会导致数据溢出，例如：long 转 int、int 转 short、short 转 byte。

  • 不能对布尔类型进行类型转换。
