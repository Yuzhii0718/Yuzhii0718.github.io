    对 Router0 进行改装，增加 WIC-1NET 模块，或者 WIC-4ESW 模块，本次使用2个 WIC-1NET 模块（WIC-4ESW 一个即可）

注意事项

  1. NAT 配置时，inside 表示内网，outside 表示外网

  2. 配置 NAT 时，inside 的 IP 地址是内网 IP 地址，outside 的 IP 地址是外网 IP 地址

  3. 多层交换机路由时，no switchport 表示将端口转换为路由端口

  4. 三层交换机双臂路由时，ip routing 表示开启路由功能

  5. 访问控制列表时，permit 表示允许，deny 表示拒绝

  6. 访问控制列表时，in 表示进入接口，out 表示出接口

  7. 访问控制列表时，access-list 后面的数字表示访问控制列表的编号
