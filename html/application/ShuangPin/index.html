使用手册

1. 切换至系统自带的英文输入法

2. 空格键 / 回车键 切换或清空

3. Tab 键 显示答案（可能不唯一）

4. 手机练习也可使用微信小程序

5. 使用 Chrome 浏览器 体验最佳

6. 查看 双拼输入介绍

关于

该应用是一个开源的双拼练习工具，旨在帮助你快速上手双拼输入法，也可在微信小程序中搜索“双拼练习”使用。

该应用的源码托管在 Github 上，该应用的开发者是 BlueSky

该应用的键位图来自 键位图，字帖来自 字帖。

这是一个被修改的版本，优化了输入体验，适配了 Windows 的输入建议，以及优化了工具栏的布局。
